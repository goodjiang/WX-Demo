### window配置

```
"navigationBarBackgroundColor": "#DCB7FF", 导航背景颜色
"navigationBarTextStyle": "black",         文字颜色 仅支持两种
"navigationBarTitleText": "小程序",         文字
"backgroundColor": "#FF827E",              下拉背景颜色
"backgroundTextStyle": "drak",             下拉加载条颜色
"enablePullDownRefresh": true,             是否启用下拉刷新
"onReachBottomDistance": 50,               上拉加载
"disableScroll": false                     垂直方向禁止滚动
```
